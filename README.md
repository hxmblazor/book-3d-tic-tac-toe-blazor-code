# Building 3D Tic-Tac-Toe with Blazor

This repository contains the final companion source code for *Building 3D Tic-Tac-Toe with Blazor: A Practical Guide to .NET 10, C# 14, and Interactive Web Development*.

The project follows a practical path from core game logic to a complete Blazor application. Along the way, it explores the same architectural and design decisions discussed in the book: separating logic from presentation, managing state carefully, scaling beyond the first requirement, and building software that stays understandable as it grows.

## What This Repository Includes

- the final companion source code aligned with the book
- the completed implementation of the 3D Tic-Tac-Toe project
- reader-facing code intended to support build-along and read-along use
- a practical Blazor example that goes beyond a small demo

## How To Use This Repository

You can use this repository in whichever way fits how you read technical books best.

- Build along chapter by chapter as you work through the book
- Compare your own implementation with the finished version
- Browse specific parts of the project when you want to study a concept in context
- Return to the completed application after finishing the book for review or experimentation

## Who This Is For

This repository is for readers who want more than a minimal sample application. It is intended for developers who want to see how a Blazor project evolves from a simple idea into a more complete application with structure, tradeoffs, and room to grow.
